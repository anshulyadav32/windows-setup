Quick Start

  1. Open PowerShell (no need for admin privileges - script will auto-elevate)

  2. Run this command to automatically install all components:

    iwr -useb https://raw.githubusercontent.com/anshulyadav32/windows-setup/main/setup.ps1 | iex
  3. The script will automatically install Windows Developer Tools, WSL2 with Ubuntu, and additional dev tools

Note: The script will automatically request administrator privileges if needed.

Direct Installation Options

If you prefer to install specific components directly:

Windows Tools Only

iwr -useb https://raw.githubusercontent.com/anshulyadav32/windows-setup/main/install.ps1 | iex

Installs essential Windows development tools and utilities.

WSL2 Ubuntu Setup

iwr -useb https://raw.githubusercontent.com/anshulyadav32/windows-setup/main/wsl.ps1 | iex

Sets up WSL2 with Ubuntu and configures it for development.

‚ö†Ô∏è Important: WSL installation requires a system restart. Run the script once, restart your computer when prompted, then run it again to complete the installation.

Additional Dev Tools

iwr -useb https://raw.githubusercontent.com/anshulyadav32/windows-setup/main/modules/devtools/install.ps1 | iex

Installs development tools, CLI utilities, and extra packages.

Note: Linux commands (like dpkg, lsb_release, etc.) must run inside Ubuntu, not in Windows PowerShell. The script will handle this automatically if you run it as shown above.

Install GitHub CLI & Notion Tools

GitHub CLI (Windows)

For GitHub command-line tools, install with:

winget install --id GitHub.cli

Notion CLI (Windows)

npm install -g @notionhq/client

WSL Installation

wsl --install -d Ubuntu

Verify Node.js Installation

npm -v

For more help, visit win.r-u.live, the GitHub repo, or the Live GitHub Pages site for issues and solutions.
