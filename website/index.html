Windows & WSL Dev Environment Installer

Automate your Windows and WSL developer setup in minutes with a single command. Automatically installs PowerShell, Ubuntu, Node.js, Python, Git, VS Code, PowerToys, CLI tools, and more.

Tech Stack: PowerShell • Bash • Node.js • Python • PostgreSQL • Docker • VS Code • CLI Tools
Quick Start View on GitHub Official Site

Quick Start

  1. Open PowerShell (no need for admin privileges - script will auto-elevate)
  2. Run this command to automatically install all components:
    iwr -useb https://raw.githubusercontent.com/anshulyadav32/windows-setup/main/setup.ps1 | iex
  3. The script will automatically install Windows Developer Tools, WSL2 with Ubuntu, and additional dev tools

Note: The script will automatically request administrator privileges if needed.

Direct Installation Options

If you prefer to install specific components directly:

Windows Tools Only

iwr -useb https://raw.githubusercontent.com/anshulyadav32/windows-setup/main/install.ps1 | iex

WSL2 Ubuntu Setup

iwr -useb https://raw.githubusercontent.com/anshulyadav32/windows-setup/main/wsl.ps1 | iex

Note: May require a system restart. The script provides step-by-step guidance.

Additional Dev Tools

iwr -useb https://raw.githubusercontent.com/anshulyadav32/windows-setup/main/modules/devtools/install.ps1 | iex
Note: Linux commands (like dpkg, lsb_release, etc.) must run inside Ubuntu, not in Windows PowerShell. The script will handle this automatically if you run it as shown above.

Troubleshooting & Fixes

  • If you see errors like npm or node not recognized, or other tools missing after install:
  • Restart PowerShell or run refreshenv to reload your environment variables.
  • Check Node.js and npm are installed:
    node -v
npm -v
  • If these commands fail, install Node.js manually or check your PATH.
  • Re-run the setup script after confirming tools are available.

For more help, visit win.r-u.live, the GitHub repo, or the Live GitHub Pages site for issues and solutions.

Advanced Installation Options

⚡ Non-Interactive Install

Install everything automatically without prompts:

iwr -useb https://raw.githubusercontent.com/anshulyadav32/windows-setup/main/setup.ps1 | iex -InstallAll
  • ✅ Fastest installation method
  • ✅ Always uses latest version
  • ✅ Automatic dependency handling
  • ✅ No user interaction required

📦 Local Installation

Clone and install locally for full control:

git clone https://github.com/anshulyadav32/windows-setup.git
cd windows-setup
.\setup.ps1
  • ✅ Full control over installation
  • ✅ Customize components
  • ✅ Modify scripts as needed
  • ✅ Works offline after cloning
